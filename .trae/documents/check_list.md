# 交付检查清单

## 视觉与风格
- [ ] 是否符合 Apple 设计规范（SF Pro 字体、16px+ 圆角、柔和阴影）
- [ ] 是否全局应用了半透明模糊 (Backdrop Blur) 效果
- [ ] 按钮与卡片的交互反馈（Hover, Active）是否自然

## 功能模块
- [ ] 酶库列表是否支持分页与筛选
- [ ] NCBI 导入任务是否能实时反馈进度
- [ ] 3D 结构查看器是否能正常渲染 PDB/CIF
- [ ] 文献匹配结果是否展示了元数据（标题、DOI、摘要）

## 接口与代码质量
- [ ] 是否已移除所有 console.log 和调试占位符
- [ ] 接口请求是否经过标准拦截器处理，包含错误提示
- [ ] 代码是否遵循 TypeScript 严谨类型定义

## 响应式适配
- [ ] 在 1920x1080 分辨率下，比例是否协调（无拉伸）
- [ ] 在 1440x900 分辨率下，侧边栏是否能优雅收缩
- [ ] 移动端（如有）是否能正常浏览核心数据
